  • Structural Framework For Dynamic Web Apps

Where You Should Use AngularJS?

  • Dynamic Applications
  • CRUD Applications
  • Client Side Solutions
  • Can make Front End Testing easier

Where You Should Not Use AngularJS?

  • Replacing jQuery
  • Static Content
  • If SEO is a Priority
  • If IE 8 is a Priority
